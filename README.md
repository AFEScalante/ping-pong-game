# Ping Pong Game

This is a simple Ping Pong game built with Pygame.

## Requirements

- Python 3.6 or higher
- Pygame

## Installation

1. **Clone the repository:**

```bash
git clone https://github.com/yourusername/ping-pong-game.git
cd ping-pong-game
```

2.	**Create a virtual environment:**

Itâ€™s a good practice to create a virtual environment to manage dependencies.

```bash
python3 -m venv venv
```

3. **Activate the virtual environment:**

* On Windows
  ```bash
  venv\Scripts\activate
  ```

* On macOS and Linux
  ```bash
  source venv/bin/activate
  ```

4. **Install dependencies:**

  Use pip to install the required packages from `requirements.txt`.

  ```bash
  pip install -r requirements.txt
  ```

## Running the game

To run the Ping Pong game, simply execute the pingpong.py script:

  ```bash
  python pingpong.py
  ```
